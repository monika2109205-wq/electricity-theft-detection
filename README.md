# ⚡ Electricity Theft Detection System

## 📌 Problem
Electricity theft causes huge losses and is difficult to detect manually.

## 💡 Solution
A smart dashboard that analyzes electricity usage and detects suspicious patterns using visualization.

## 🚀 Features
- 🔐 Login Authentication
- 📊 Daily, Weekly, Monthly Usage Graphs
- 🚨 Suspicious Activity Detection
- 📋 Alerts Dashboard

## 🛠️ Tech Used
- HTML, CSS, JavaScript
- Chart.js

## 🎯 Future Scope
- AI-based prediction
- Real-time smart meter integration
